EMPLOYEE ATTENDANCE SYSTEM


Project Info

The "Employee Attendance System" is a web-based application designed to streamline and automate daily attendance tracking within an organization. Employees can securely log in, mark their attendance by clocking in and out, and submit leave requests directly through the system. The admin has access to a centralized dashboard where they can manage employee records, approve or reject leave applications, and export attendance data in CSV format for reporting purposes. The system incorporates real-time timers, flash messaging for user feedback, and intuitive form validations to enhance usability. This project was developed using PHP for the backend logic and MySQL for data storage, ensuring robust session handling and accurate data retrieval. The frontend is built with HTML, CSS, and Bootstrap 5 to provide a clean and responsive user interface, while JavaScript handles client-side interactivity such as live timers and form dynamics. Overall, the system improves transparency, reduces manual effort, and ensures consistent tracking of employee working hours and leaves.

Skill Used

HTML, CSS, Bootstrap, PHP, MySQL
