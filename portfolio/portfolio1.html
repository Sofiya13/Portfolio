Project Info

With each click, the selected color instantly updates the background, providing a visually dynamic experience. The interface is clean and responsive, making it perfect for users who want a quick and fun way to customize the look of a webpage or application.

BMI GENERTOR


Project Info

This project is an easy-to-use "BMI Generator" that allows users to calculate their Body Mass Index (BMI) based on their height and weight. Users can input their measurements, and the tool quickly computes the BMI, displaying the result along with a corresponding health classification (underweight, normal, overweight, or obese). The design includes a reset button, enabling users to clear their inputs and start fresh. With a clean and intuitive interface, this project is perfect for anyone looking to track their health and understand their body composition in a straightforward manner.

EMERGENCY HOSPITAL


Project Info

This project is a simple and efficient discount calculator that takes user input for the original amount and the percentage of discount. It then calculates and displays the final price after applying the discount. The design includes a reset button, allowing the user to easily reset the input values back to zero for a fresh calculation. The interface is clean and user-friendly, focusing on functionality and simplicity, making it perfect for anyone looking to quickly determine the discounted price of an item.
