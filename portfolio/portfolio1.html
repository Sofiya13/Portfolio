BMI GENERTOR


Project Info

This project is an easy-to-use "BMI Generator" that allows users to calculate their Body Mass Index (BMI) based on their height and weight. Users can input their measurements, and the tool quickly computes the BMI, displaying the result along with a corresponding health classification (underweight, normal, overweight, or obese). The design includes a reset button, enabling users to clear their inputs and start fresh. With a clean and intuitive interface, this project is perfect for anyone looking to track their health and understand their body composition in a straightforward manner.

EMERGENCY HOSPITAL


Project Info

This project, "Emergency Hospital" is a web application designed to help users quickly find hospitals near their location in case of an emergency. By leveraging HTML, CSS, and JavaScript, the application provides a user-friendly interface where users can input their current location and the type of medical emergency they are experiencing. The application then uses geolocation and mapping APIs to display a list of nearby hospitals that specialize in treating the specified condition. This tool is invaluable for individuals needing urgent medical attention, as it significantly reduces the time spent searching for appropriate healthcare facilities, ensuring faster access to critical care.
